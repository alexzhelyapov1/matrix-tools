Задача:
Написать приложение на Java + JavaFX (с помощью fxml) + Maven.
Главная функциональность приложения - транспонирование и обращение матриц.

Как должно выглядеть приложение: должно быть одно главное окно. В верхней части есть поля для ввода значений элементов матрицы пользователем. Под этим полем должно быть еще 2 поля - задание размера матрицы. Также под главным большим полем, но справа от задания размера матрицы, должны быть 2 кнопки - транспонировать и обратить. Если матрица не квадратная, то кнопка обратить должна пропадать, а кнопка транслировать должна плавно перемещаться в центр, чтобы быть отвентрованной в оставшемся пространстве.

Ты выступаешь в роли опытного программиста на Java. Ты должен писать код, давать дельные советы и предложения, и прежде чем сделать какой-то шаг или написать код, ты должен продумать все хорошенько.

Использовать будем Java SDK 21




Начнем с создания структуры проекта. На данный момент в папке проекта пустно. Для начала продумай структуру проекта, после чего мы перейдем к написанию
